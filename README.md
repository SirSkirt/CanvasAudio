# Canvas Audio (Modular)

Modularized version of the single-file Canvas Audio Tone.js web DAW.

## Run locally
Because ES modules are used, serve via a local web server (not `file://`):

- VS Code: Live Server
- Python: `python -m http.server 8000`

Then open `http://localhost:8000/`.

## Deploy to GitHub Pages
Push this folder to a repo and enable Pages (Deploy from branch).
